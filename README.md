A personal research project focused on building a Large Language Model (LLM) from the ground up using Python. This project aims to explore the mathematical foundations and core architectures of modern transformer-based language models through hands-on implementation.

---

## 🚧 Status: Work in Progress
This project is currently **under active development**. Many core components are implemented, but the model is being refined continuously. Contributions and feedback are welcome!

---

## 🔍 Project Goals
The main objective is to implement an LLM from scratch without relying on high-level black-box frameworks for the core logic. Key areas include:
* **Custom Tokenization**: Designing and implementing efficient tokenization strategies.
* **Transformer Blocks**: Building multi-head attention, feed-forward networks, and normalization layers from scratch.
* **Training Pipeline**: Developing a robust training loop with custom loss functions and optimization strategies (gradient clipping, learning rate scheduling).

---

## 🛠️ Technical Stack
* **Language**: Python
* **Core Logic**: Implementing transformer architectures using core mathematical operations.
* **Current Focus**: Finalizing the transformer block architecture and model training loop.
